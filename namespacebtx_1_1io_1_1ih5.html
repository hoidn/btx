Decorator mainly for write functions.

Will not allow the decorated function to execute if an associated '.lock' file exists. If no lock file exists one will be created, preventing other decorated functions from executing while the current function completes its operations. Attempts at function execution will also be abandoned after a timeout threshold is reached.

Usage e.g.: @lock_file('shared.h5', timeout=2) def my_write_function(): print('Writing file safely') my_write_function() # Create shared.lock if it doesn't exist

Parameters
path(str) Path including filename of the file to be locked.
timeout(int) The timeout time in seconds to abandon execution attempt.
wait(float) Time in seconds to wait between write attempts.
